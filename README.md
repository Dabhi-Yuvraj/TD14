# JavaScript Array Operations

This project demonstrates basic JavaScript functions that perform different operations on an array.

## Features

- Find the maximum number in an array using a normal function.
- Calculate the sum of all elements using an arrow function.
- Count the number of odd elements using a callback function.
- Display all results using `console.log()`.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Press **F12** and open the **Console** tab.
4. View the output of all array operations.
